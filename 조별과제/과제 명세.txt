파일 목록
 1. client
 ㄴ 고객 정보 저장
 ㄴ 학번 (정수 8자리)
  ㄴ 서로 다른 값 가져야됨. 중복된 값 x
 ㄴ 비밀번호 (문자열)
 ㄴ 이름 (문자열)
 ㄴ 주소 (문자열)
 ㄴ 전화번호 (문자열)
 ㄴ 학번으로 정렬
 2. book
 ㄴ 도서 목록 저장
 ㄴ 최소 100권 이상의 정보 포함 필요
 ㄴ 도서 번호 (정수 7자리)
  ㄴ 서로 다른 값 가져야됨. 중복된 값 x
 ㄴ 도서명 (문자열)
 ㄴ 출판사 (문자열)
 ㄴ 저자명 (문자열)
 ㄴ ISBN (정수 13자리)
 ㄴ 소장처 (문자열) // 비치된 곳 의미
 ㄴ 대여가능 여부 (문자)
  ㄴ N : 대여 가능 여부
  ㄴ Y : 그렇지 않은 경우
 ㄴ ISBN 숫자로 정렬
 3. borrow
 ㄴ 대여 목록 저장
 ㄴ 최소 20건 이상의 정보 포함 필요
 ㄴ 학번 (정수 8자리)
 ㄴ 도서번호 (정수 7자리)
 ㄴ 대여일자 (time_t형)
 ㄴ 반납일자 (time_t형)
  ㄴ 대여일자 + 30
   ㄴ 30일째 되는 날이 일요일이면 대여일자 + 31

프로그램 실행하면 즉시 파일 정보 메모리에 로드

파일 양식은
내용1 | 내용2 | 내용3 | ...

파일의 내용이 변경되는 경우 즉시 파일 갱신해야됨

동적 할당, 연결 리스트 사용 필수

프로그램
 1. 회원용
  (1) 도서검색
  (2) 자신의 대여목록
  (3) 개인정보수정
  (4) 회원탈퇴
 2. 관리자용
  (1) 도서등록
  (2) 도서삭제
  (3) 도서대여
  (4) 도서반납
  (5) 도서검색
  (6) 회원목록

초기 메뉴
 1. 회원가입
 2. 로그인
 3. 프로그램종료

회원가입
 1. 학번, 비밀번호, 이름, 주소, 전화번호를 입력받아 client에 저장
 2. 동일한 학번이 있으면 오류메시지 출력
 3. 학번순으로 정렬하여 저장
 4. 완료되면 초기 메뉴로 돌아감

로그인
 1. 로그인명, 비밀번호 입력
 2. 학번으로 입력되면 회원용
 3. admin이 입력되면 관리자용

회원용
 1. 도서 검색
 ㄴ 도서 정보 출력
  ㄴ도서명, 출판사, 저자명, ISBN, 소장처, 대여가능 여부 출력
  (1) 도서명 검색
  (2) 출판사 검색
  (3) ISBN 검색
  (4) 저자명 검색
  (5) 전체 검색
  ㄴ book에 있는 모든 도서의 정보 출력
  (6) 이전 메뉴
  ㄴ 회원 메뉴로 리턴
 2. 내 대여 목록
 ㄴ 도서번호, 도서명, 대여일자, 반납일자 출력
 3. 개인 정보 수정
 ㄴ 학번과 이름을 제외한 모든 정보 수정
 ㄴ client 파일에 저장
 4. 회원 탈퇴
 ㄴ 대여 중인 도서 목록이 없을 때만 가능
 ㄴ client에 해당 회원의 정보 삭제
 5. 로그아웃
 ㄴ 초기 화면으로 이동
 6. 프로그램 종료

관리자용
 1. 도서 등록
 ㄴ 도서명, 출판사, 저자명, ISBN, 소장처 입력
 ㄴ 대여 가능 여부와 도서 번호는 자동으로 처리
 ㄴ 대여 가능 여부는 Y
 ㄴ 마지막으로 등록된 도서의 도서 번호 + 1
 ㄴ book 파일에 저장
 2. 도서 삭제
 ㄴ 먼저 도서 검색
  ㄴ 도서명 검색 or ISBN 검색으로 삭제할 도서 검색
  ㄴ 검색 결과 : 도서 번호, 도서명, 출판사, 저자명, ISBN, 소장처, 삭제 가능 여부
   ㄴ 대여 가능 여부가 N이면 삭제 가능 여부는 N // 대여 중이면 삭제 ㅂㄱㄴ
   ㄴ 대여 가능 여부가 Y이면 삭제 가능 여부는 Y
 ㄴ 삭제 시 검색된 도서 번호를 입력
  ㄴ 대여 중인 경우 삭제 ㅂㄱㄴ
 ㄴ book 파일에 저장
 3. 도서 대여
 ㄴ 먼저 도서 검색
  ㄴ 도서명 검색 or ISBN 검색으로 대여해 줄 도서 검색
  ㄴ 검색 결과 : 도서 번호, 도서명, 출판사, 저자명, ISBN, 소장처, 대여 가능 여부
 ㄴ 대여일자, 반납일자 저장
 ㄴ 대여 가능 여부를 N으로 변경
 ㄴ book 파일과 borrow 파일에 저장
 4. 도서 반납
 ㄴ 먼저 학번을 입력
 ㄴ 해당 회원의 모든 도서 대여 목록 출력
 ㄴ 도서 번호를 입력하면 도서 반납 처리 됨
  ㄴ borrow 파일에서 도서 대여 정보 삭제
  ㄴ 대여 가능 여부를 Y로 변경
 ㄴ book 파일과 borrow 파일에 저장
 5. 도서 검색
 ㄴ 회우너 서비스의 도서 검색과 동일 // 동일 함수 사용이 좋을듯
 6. 회원 목록
  (1) 이름 검색
  ㄴ 해당 회원의 학번, 이름, 주소, 전화번호 정보가 출력
  (2) 학번 검색
  ㄴ 해당 회원의 학번, 이름, 주소, 전화번호 정보가 출력
  (3) 전체 검색
  ㄴ client에 있는 정보 출력
  (4) 이전 메뉴
 7. 로그아웃
 8. 프로그램 종료
